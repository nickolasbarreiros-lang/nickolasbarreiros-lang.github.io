# Sprint 3 — Ficha visual V3

Esta atualização melhora a apresentação das fichas sem duplicar código.

## Arquitetura preservada

- Cada orquídea continua em seu próprio módulo dentro de `dados/orquideas/`.
- `ficha.js` permanece como o único renderizador visual compartilhado.
- Alterações futuras de layout continuam sendo feitas em um único ponto.

## Melhorias implementadas

- Navegação fixa por seções.
- Âncoras com rolagem suave.
- Selo Serra/ES.
- Acabamento visual do cabeçalho e dos cartões.
- Indicação de ampliação nas fotografias.
- Botão flutuante de retorno ao topo.
- Ajustes responsivos para celular.
